Metsi Technologies

The client wanted full access to the CMS, so its usability was very important. I built in extra functionality so that the client could decide the width of the hero image for each page with the toggle of a switch. All images, logos and text were pulled in from WordPress, so the client could add and change what they wanted.

Features like this were important to them because they were supplying content, and wanted to add a lot after coding had finished. Hidden fields in the their vacancy application form were also built in to automatically tell the client which job the user was applying for without the user having to specify themselves.

The Viceroy

This design presented some interesting challenges for me to code. The images that run down either side of the pages had to be strategically placed to sit well on the page on different screen sizes. I also wanted the client to be able to change the images in the backend of WordPress if they wanted to, so the pattern had to be overlaid separately. This meant that the image will always be a full image that is resized appropriately and can be changed whenever the client wishes.

There are a few design touches that I had to think about how to code them, including the offer the sits half on the hero and half below, and the patterns that sit either side of the page. The map is also a custom map to ensure the colours and layout matched the design.

Created at Adtrak

Manor Mix Concrete

The content on this site is quite for down the page, which meant a lot of scrolling on smaller devices. To make usability easier, I made the sections above the content toggle (Why Choose Us and the calculator). The calculator also links through to the contact form, using the calculated result to populate the appropriate input field.

The client also wanted a sticky/fixed header but with two navs, it really starts to reduce the user’s view. I created a sticky header but made sure that when you start to scroll the height of the header reduces.

Created at Adtrak

Sussex Ready Mixed Concrete

This was a relatively small site that I designed and built efficiently for a quick launch for the client. The challenge was incorporating their original logo into a fresh design. The dark grey and green are mixed with different tones to pull them together.

The site features a concrete calculator that takes the result and inserts it into the contact form if the user chooses to click through to order online.

Animation

This is the final piece of my Fine Art degree. The project started with paintings that focused on movement and expression. These initial paintings were laid out on the floor so that I could find links between them. I made more paintings to fill in the gaps so that it resulted in a series of stills where the lines on each painting followed onto the next. The paintings were photographed and manipulated and turned into an animation using Photoshop and After Effects.
