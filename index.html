Personal Profile

I am currently working in the Web Design department for Adtrak, a large digital marketing agency based in Nottingham. My passion lies with coding and developing websites but I am heavily involved with projects from initial conception, right through the design stage, coding, testing and setting the site live. WordPress is the CMS that I use to create dynamic websites. I have expanded my PHP skills by building on these sites and coding in extra functionality for the clients to use in the backend of WordPress.

I have used JavaScript and jQuery to bring interactivity to them, and it is a language that I am looking to explore a lot more. I have thoroughly enjoyed working with JavaScript; from creating custom maps with the Google Maps API, to combining it with WordPress ACF Fields to create tabs that the client can control from the backend of WordPress, and customising forms by populating inputs with information about certain pages that will be helpful for the client receiving the enquiry.

Styling is done using SASS, and I am starting to look more into the BEM methodology to improve readability and usability of my code. I am also comfortable dealing with MySQL databases, using CPanel and WHM to manage the sites.

In my personal time, I enjoy learning about new technology that I can use in projects. I am currently improving my skills with JavaScript by looking into frameworks such as react.js and node.js. I write a blog to document some of the things that I am using. This helps me to digest and remember the information, and also possibly share solutions to problems that I have overcome to others that may be experiencing similar problems.

Metsi Technologies

The client wanted full access to the CMS, so its usability was very important. I built in extra functionality so that the client could decide the width of the hero image for each page with the toggle of a switch. All images, logos and text were pulled in from WordPress, so the client could add and change what they wanted.

Features like this were important to them because they were supplying content, and wanted to add a lot after coding had finished. Hidden fields in the their vacancy application form were also built in to automatically tell the client which job the user was applying for without the user having to specify themselves.

Chattrbox Web Application

I built this web app with the help of a front-end development book. I used Node.js to build a WebSockets server to facilitate real-time communication. Npm scripts were used to streamline my workflow and it also gave me a chance to work more from the terminal. ES6 JavaScript was compiled into ES5 using Babel and Browserify so that it can be used on older browsers.

This was a great project that allowed me to get a more in depth look into the workings of an application. I was able to use a modular approach and get to grips with features that I was more unfamiliar with like JavaScript classes, arrow functions and imports and exports.

My next job for this project is to improve the design and experiment with some animations and effects for when new messaged appear on the screen.

Animation

This is the final piece of my Fine Art degree. The project started with paintings that focused on movement and expression. These initial paintings were laid out on the floor so that I could find links between them. I made more paintings to fill in the gaps so that it resulted in a series of stills where the lines on each painting followed onto the next. The paintings were photographed and manipulated and turned into an animation using Photoshop and After Effects.
